## 📌 Sobre o Projeto

Essa página foi criada com o objetivo de praticar:

- Estruturação semântica com HTML
- Construção de layout moderno
- Estilização com CSS
- Organização de menu de navegação
- Alinhamento de elementos
- Uso de botões estilizados
- Hierarquia visual

O design simula uma landing page para serviços de cuidados com pets.
